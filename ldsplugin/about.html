QGIS LDS Plugin

The QGIS LDS Plugin couples QGIS an d the LINZ Data Services (LDS)  to provide simple importing of LINZ data. 

Selecting a Service 

The QGIS LDS Plugin supports LDS data served over WFS, WMS, WMTS protocols. 

Please see the online resource  available on the LINZ website for more on these protocols 

API Key 

Prior to using the plugin your LDS API Key must be stored via the "Setting menu"

For more on LDS API keys please see the LDS resources.

Coordinate Reference System

When selecting and importing LDS datasets via the QGIS LDS Plugin the projects current Coordinate Reference System (CRS) will be used. QGIS will enable on-the-fly (OTF) Reprojection and the dataset being imported will be reprojected on the fly to the QGIS projects CRS. 

This however is not the case with WMTS. As it is not idea to reproject WMTS and all LDS WMTS services are served in WGS 84 Web Mercator (EPSG:3857), the QGIS LDS Plugin will force the QGIS projects CRS to WGS 84 Web Mercator (EPSG:3857). The user is notified by QGIS's message bar when the CRS change is made. 

As many datasets are being reprojected to be combined into the one map canvas the user of this plugin should be aware of issues arising from projecting spatial datasets. 

Requests, Responses and Patience

When opening the plugin for the first time, the QGIS interface will be temporarily inactive while awaiting the LDS server's response. However after the initial request LDS dataset information is stored locally and all subsequent requests for the current QGIS session will see the dataset information populated immediately. 

At times the LDS server is experiencing high load the plugin may fail to populate the table with the available dataset information. In this case the user will need to close the plugin interface and re-open it. 

Filtering

The left hand panel allows users to filter by service types (either, All, WFS, WMS, WMTS). All column headers can be toggled to allow ascending or descending ordering of their data.

Text can be entered in The "Filter Data Sets" search bar to filter the datasets on key words.
